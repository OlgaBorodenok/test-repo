Чтобы добавить картинку в ветку Git:

• скопируй файл изображения в локальную папку репозитория
• используй команду git add <имя_файла> для добавления файла в индекс. 
• сделай коммит с помощью команды git commit -m "Добавил картинку",
• отправь изменения в удаленный репозиторий командой git push




Чтобы проверить, добавились ли остальные файлы, убедись, что ты выполнила команды 
git add .
git commit -m "your message" 
и git push

После этого обнови страницу репозитория на GitHub, и ты должна увидеть все добавленные файлы.

Чтобы удалить репозиторий на GitHub,
 зайди в настройки (Settings) нужного репозитория. 
Пролистай страницу вниз до раздела "Danger Zone" и нажми "Delete this repository". 
Тебя попросят подтвердить удаление, введя название репозитория.
Будь внимательна, так как это действие необратимо.
